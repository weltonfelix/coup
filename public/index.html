Controle da Partida

/iniciar: Inicia a partida

Ações de turno

/renda: Pega 1 moeda

/ajudaextra: Pega 2 moedas

/duque: Pega 3 moedas

/roubar {jogador alvo}: Rouba 2 moedas de um jogador alvo

/pagar {x}: Paga uma quantidade x de moedas

/dropassassinato {nome carta}: Escolhe uma carta para perder após ser assassinado pelo Assassino

/dropgolpe {nome carta}: Escolhe uma carta para perder após levar um golpe de estado

/embaixador: Pega 2 novas cartas do deck

/trocar {nome carta}: Troca uma carta sua por outra aleatória do baralho

/devolver {nome carta}: Devolve uma carta sua para o baralho
